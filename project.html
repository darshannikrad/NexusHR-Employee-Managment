About NexusHR

A serverless, cloud-native employee management system built on AWS

👋 About the Builder

Hi, I'm Yash Nimbalkar, a third-year engineering student at PICT, Pune. I built NexusHR to showcase my AWS cloud skills — connecting a serverless backend with a modern frontend to create a real-world employee management system.

🏗️ Architecture

Fully serverless on AWS — no servers to manage, no idle costs. Scales automatically with demand.

AWS Lambda DynamoDB S3 API Gateway Python HTML/CSS/JS

✨ Features

Employees are stored in DynamoDB with auto-generated IDs like EMP-EN-2025-AB12. Profile photos are stored in S3. CRUD operations exposed via REST API Gateway endpoints. Auth with localStorage sessions.

🗺️ Frontend Stack

HTML5 CSS3 JavaScript ES6+ Hosted on S3

Lightweight, dependency-free frontend for fast loads. Light & dark themes. Drag-and-drop image upload. Session-based auth.

🚀 Future Scope

Plans include: department filters, edit employee details, pagination, authentication with AWS Cognito, analytics dashboards and export to CSV.
