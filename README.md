## ✨ About The Project

NexusHR is a modern employee management system built using a serverless AWS architecture and a lightweight frontend deployed on Netlify.

It demonstrates real-world cloud concepts like:
- ⚡ Serverless computing  
- ☁️ Scalable backend  
- 🌐 API-driven architecture  
- 🎯 Clean UI/UX  

---

## 🧠 Architecture

User → Netlify (Frontend)  
     ↓  
 API Gateway (AWS)  
     ↓  
 Lambda Functions  
     ↓  
 DynamoDB + S3  

---

## 🛠️ Tech Stack

### 🌐 Frontend
- HTML5  
- CSS3  
- JavaScript (ES6)  
- LocalStorage (Session Auth)

### ☁️ Backend (AWS)
- AWS Lambda  
- API Gateway  
- DynamoDB  
- Amazon S3
